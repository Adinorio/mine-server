## Option 3: Get Router Password (Best Long-term)

**Once you have it, port forwarding is perfect!**

### Ways to get password:
1. **Ask the router owner** nicely
   - Explain you need it for gaming
   - Offer to change it back after

2. **Check router sticker**
   - Default password might be there
   - Try common defaults: `admin`, `password`, `1234`

3. **Reset router** (if you have physical access)
   - Find reset button (small hole)
   - Hold 10-30 seconds
   - Default password on sticker
   - **Warning:** Resets WiFi password too!

### Once you have access:
- Port forwarding is free, permanent, reliable
- No software needed for anyone
- Best solution overall
